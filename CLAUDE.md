## Development Commands

```sh
bun run dev             # Start development server
bun run check           # Type-check with svelte-check
bun run lint            # Run prettier and eslint
bun run format          # Format code with prettier
bun run build           # Build SvelteKit application
```

## Documentation

The project includes a Docusaurus documentation site at `documentation/docs/`:

```sh
cd documentation
npm run start           # Start documentation dev server
npm run build           # Build documentation
```
